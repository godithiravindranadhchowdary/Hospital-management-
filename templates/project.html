🏥 Hospital Management System

A Comprehensive Healthcare Platform

📋 About This Project

The Hospital Management System is a comprehensive healthcare management solution built with Django and Django REST Framework. It provides a complete platform for managing appointments, prescriptions, patient records, billing, and more.

✨ Key Features

  • Role-based authentication (Admin, Doctor, Patient)
  • Appointment scheduling with conflict detection
  • Digital prescription management
  • Invoice and payment tracking
  • Doctor leave management
  • Medical records management
  • Real-time dashboard analytics
  • RESTful API with JWT authentication

🛠️ Technology Stack

🐍 Django
⚡ DRF
🟨 JavaScript
🎨 CSS3
🐳 Docker
☁️ Render

👥 User Roles

  • Admin: Full system access, manage doctors, patients, view analytics
  • Doctor: Manage appointments, prescriptions, view patients
  • Patient: Book appointments, view prescriptions, make payments
